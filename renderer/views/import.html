Information:

Compared to other key gen software, RTS EM uses hyphens "-" instead of underscores "_" to separate key types
RTS EM also uses the keyword "private" instead of "secret" to designate the private key.

To import keys please modify your keys file names where the public key has either "-public" or "-pub" at the end,
where as the private key has either "-private" or "-pvt" at the end respectively


Public: ExampleKey-public.key
Private: ExampleKey-private.key
